# Agent Guide

Use this guide when operating the CCFA family as an agent.

1. Read only the owning skill's `SKILL.md` first.
2. If routing is ambiguous, check `ccf-common/references/skill-trigger-registry.yaml`.
3. If the task touches project state, read `ccfa.yaml` and `ccf-common/references/artifact-contracts.md`.
4. If the task names a venue, read `ccf-writing-skills/references/venue-guides/index.md` before the specific venue guide.
5. If official venue policy matters for final submission, verify the current official page before making a final claim.
6. Do not let audit skills rewrite content unless the user explicitly asks and routing permits the handoff.
