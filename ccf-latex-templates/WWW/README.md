# WWW (The Web Conference)

WWW uses the standard ACM `acmart.cls` template.

## Template

See the `../ACM/` folder for the full `acmart.cls` package.

## Usage

```latex
\documentclass[sigconf]{acmart}
```

For review mode (anonymous submission):
```latex
\documentclass[sigconf, review]{acmart}
```

For camera-ready final version:
```latex
\documentclass[sigconf]{acmart}
```

## Source

The ACM template is maintained at: https://github.com/borisveytsman/acmart
